# Periodic-Table-Search-Program
It is a periodic table search program using c++ without oops. Easy and useful for students.

A C++ console application that allows users to search for elements in the periodic table by entering the element's name. The program displays comprehensive details about the element, making it both educational and interactive.

Features

Search an element by name

Display important properties:

Name

Symbol

Atomic Number

Atomic Mass

Category (e.g., Metal, Nonmetal, Metalloid)

Group & Period

Fun Fact

User-friendly console interface

Quick and accurate element lookup

How It Works

Run the program in a C++ compiler.

Enter the name of an element.

View the detailed properties of the element instantly.

Example Output
Enter element name: Oxygen

Name: Oxygen
Symbol: O
Atomic Number: 8
Atomic Mass: 15.999
Category: Nonmetal
Group: 16
Period: 2
Fun Fact: Oxygen is essential for respiration and supports combustion.

Technologies Used

C++

Standard Input/Output (iostream)

Arrays / Structs for storing element data
